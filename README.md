# Sculpt Mask Manager

A minimalist sculpt mask management tool for Blender 4.2+.

## Features
- **Save/Load Masks**: Save current sculpt mask as vertex groups and load them back.
- **High Performance**: Uses NumPy for fast data processing on high-poly meshes.
- **Minimalist UI**: Access via N-panel (Sculpt tab) or Alt+M Pie Menu.
- **Blend Modes**: Click to replace, Shift+Click to add, Ctrl+Click to subtract.

## Installation
1. Download the `sculpt_mask_manager.zip`.
2. In Blender, go to **Edit > Preferences > Get Extensions**.
3. Click the arrow in the top right and select **Install from Disk...**.
4. Select the zip file.

## Usage
- **Alt + M**: Open the Pie Menu at mouse position.
- **N-Panel**: Find "Mask Groups" under the "Sculpt" tab.
